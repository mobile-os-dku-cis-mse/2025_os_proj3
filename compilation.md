Requirements

A Unix-like operating system (Linux or macOS recommended)

GCC or any compatible C compiler

Standard C libraries (stdio.h, stdlib.h, string.h...)

The project header file fs.h

A valid filesystem disk image file (disk.img) located in the project directory

The compilation follows this pattern:
gcc -o (name_you_want) fs.c

./(name_you_want) /(name_of_the_file_you_want_to_read)
